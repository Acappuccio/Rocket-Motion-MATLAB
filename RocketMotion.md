# `RocketMotion`
* Model and simulate a rockets behavior of flight path taking into consideration the physical forces acting upon the rocket. 
## Syntax 
```matlab
function[next]=RocketMotion(current,dt)
```
## Description 
* The `RocketMotion` function uses newtons laws of motion to calculate the state of the rocket `current` along with the change in  time `dt 
to predict the rockets trajectory `next` .
